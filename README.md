# AI-Powered Banking Fraud Detection

This project detects fraudulent banking transactions using Machine Learning.

## 📌 Problem Statement
Traditional rule-based fraud detection systems fail to detect evolving fraud patterns. This project builds a predictive ML model to identify fraudulent transactions using transaction patterns.

## 🚀 Tech Stack
- Python
- Scikit-learn
- XGBoost
- Pandas
- SMOTE

## 📊 Models Used
- Logistic Regression
- Random Forest
- XGBoost

## 🎯 Evaluation Metrics
- Precision
- Recall
- F1 Score
- ROC-AUC

## 📁 Project Structure
- src/ → Model training & preprocessing
- notebook/ → EDA & experimentation
- models/ → Saved trained model
- app.py → Deployment script

## 📈 Outcome
A trained ML model that predicts fraudulent transactions based on transaction features.
